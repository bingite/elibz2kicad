// 9 顶层装配层 -> F.Fab

// 2 底层 -> B.Cu
// 4 底层丝印层 -> B.Silkscreen
// 6 底层阻焊层 -> B.Mask
// 8 底层锡膏层 -> B.Paste
// 10 底层装配层 -> B.Fab

// 11 板框层 -> Edge.Cuts
// 12 多层 -> F&B.Cu *.Mask
// 13 文档层 -> User.Drawings
// 14 机械层 -> null
// 56 钻孔图层 -> null
// 57 飞线层 -> null

// 48 元件外形层 -> F.Fab
// 49 元件标识层 -> User.7
// 50 引脚焊接层 -> User.8
// 51 引脚悬空层 -> User.9
